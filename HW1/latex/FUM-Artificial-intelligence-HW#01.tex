\section{سوال اول}
در بحث \lr{Agent Program}، چهار نوع عامل معرفی شدند:
\begin{itemize}
    \item \lr{Simple Reflex Agent}
    \item \lr{Model-Based Reflex Agent}
    \item \lr{Goal-Based Agent}
    \item \lr{Utility-Based Agent}
\end{itemize}
\noindent
شبه‌کد مربوط به \lr{Goal-Based Agent} را بنویسید که بر اساس وضعیت محیط، قوانین و اهداف خود تصمیم به انجام عمل مناسب می‌گیرد. 
همچنین دیاگرام آن را نیز رسم کنید.  
سپس شبه‌کد \lr{Utility-Based Agent} را بنویسید که بر اساس بیشینه‌سازی  \lr{Utility} در وضعیت‌های ممکن، عمل مناسب را انتخاب می‌کند. 
همچنین دیاگرام آن را نیز رسم کنید.

\section{سوال دوم}
یک \lr{state space} را در نظر بگیرید که \lr{state} شروع آن عدد \(1\) است  و هر state به شکل \(k\) دو \lr{successor} دارد: اعداد \(2k\) و \(2k+1\).

\begin{enumerate}
    \item بخش مربوط به state های \(1\) تا \(15\) را رسم کنید.
    \item فرض کنید \(\)  state هدف برابر \(15\) است. ترتیب بازدید از گره‌ها را برای هر یک از روش‌های زیر بنویسید (فرض کنید در هر گره ابتدا \lr{Left} یعنی \(2k\) و سپس \lr{Right} یعنی \(2k+1\) بررسی می‌شود):
    \begin{enumerate}
        \item \lr{Breadth-First Search}
        \item \lr{Depth-Limited Search} با حد (limit) برابر \(3\)
        \item \lr{Iterative Deepening Search}
    \end{enumerate}
    \item روش \lr{Bidirectional Search} در این مسئله چقدر خوب کار می‌کند؟ ضریب \lr{branching factor} را در هر جهت (از ریشه به هدف و از هدف به ریشه) مشخص کنید و توضیح کوتاهی دربارهٔ کارایی آن بنویسید.
\end{enumerate}

\section{سوال چهارم}
معیارهای ارزیابی یک الگوریتم جستجو به صورت چهار مورد زیر می‌باشد:
\begin{itemize}
    \item \textbf{ Completeness :} آیا الگوریتم تضمین می‌کند که وقتی راه‌حلی وجود دارد آن را پیدا کند و وقتی راه‌حلی وجود ندارد، درست اعلام کند که راه‌حلی نیست؟
    \item \textbf{ Cost Optimal :} آیا الگوریتم راه‌حلی را پیدا می‌کند که کمترین هزینه مسیر را در بین همه راه‌حل‌ها داشته باشد؟
    \item \textbf{ Complexity Time :} الگوریتم برای پیدا کردن راه‌حل چقدر زمان لازم دارد؟
    \item \textbf{ Complexity Space :} الگوریتم چه مقدار حافظه برای انجام جستجو نیاز دارد؟
\end{itemize}
الگوریتم‌های زیر را بر اساس چهار معیار بالا ارزیابی کنید:
\begin{itemize}
    \item (BFS) Search Breadth-First
    \item (UCS) Search Uniform Cost
    \item (DFS) Search Depth-First
    \item (DLS) Search Depth-Limited 
    \item (IDS) Search Iterative Deepening
    \item Search Bidirectional 
\end{itemize}
بهتر است اطلاعات خود را به صورت جدول سازماندهی کنید. همچنین مقادیر \textbf{ Complexity Time } و \textbf{ Complexity Space} هر الگوریتم نیز بیان شود.
